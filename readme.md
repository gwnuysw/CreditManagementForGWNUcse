# 학점 관리 시스템

강릉원주대학교 권기태 교수님의 소프트웨어공학 팀프로젝트입니다.

**학점 관리 시스템의 주요 기능**

- 학생이 그동안 어떤 강의을 수강했는지 알 수 있다.
- 현재 까지 몇 학점을 이수 했는지 알수 있다.
- 졸업 까지 몇 학점을 더 이수 해야하는지 알 수 있다.
- 이번 학기 어떤 강의를 수강해야 하는지 알 수 있다.
